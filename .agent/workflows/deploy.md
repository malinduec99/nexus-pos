description: How to synchronize and deploy changes to MEC ERP (Web & Desktop)

Whenever any change is made to the codebase (HTML, CSS, JS, etc.), this script MUST be run to ensure the system is in sync across all platforms.

// turbo
1. Run the master sync command: `.\MASTER_SYNC.bat`
2. Follow the on-screen prompts for Electron publishing if version changes are detected.
